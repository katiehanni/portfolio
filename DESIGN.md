# Design: Clean Signal
**Date:** 2026-06-12 · **Status:** confirmed
**Archetype:** Sage (inflected with Caregiver warmth) · **Register:** brand
**DNA:** Swiss base + type voice from Soft Futurism · **Dominant axis:** layout discipline

## Direction
A crisp, grid-ruled portfolio that speaks fluently to both technical and business audiences — the same rare combination Katie brings to every role. White/grey chrome with a single sage-green accent signals precision without coldness; the humanist sans voice keeps it approachable rather than sterile. Every layout decision earns its place via hierarchy and placement alone, no decorative scaffolding.

## Signature move
The sage-green accent (#59d299) appears in exactly two places: the active section indicator in the sticky nav, and the primary CTA ("View Resume" / "Contact"). Every other UI element — borders, secondary text, surfaces — is neutral grey. This rations the color so it reads as a signal, not decoration.

## Type
- Display + Body: Instrument Sans (Google Fonts) — `font-family: 'Instrument Sans', system-ui, sans-serif`
- Scale: 1.25 (Major Third) from 16px base
  - xs: 12px · sm: 14px · base: 16px · md: 20px · lg: 25px · xl: 31px · 2xl: 39px · 3xl: 49px
- Leading: 1.6 body · 1.1 display headings
- Weights: 400 (body), 500 (labels/nav), 600 (subheadings), 700 (display)
- Labels: uppercase, 0.08em letter-spacing, font-size 12px, weight 500
- Numerals: tabular where data appears (skill bars, years, metrics)

## Space, shape, depth
- Spacing scale (4px base): 4 · 8 · 12 · 16 · 24 · 32 · 48 · 64 · 96 · 128
- Radius: 0px everywhere except pill tags (999px) — no rounded cards
- Borders: 1px solid var(--border) — hairlines only, no filled card backgrounds
- Shadows: none on layout; only on the sticky nav (box-shadow: 0 1px 0 var(--border))
- No pure #000 or #fff — use tokens exclusively

## Motion
- Micro: 100ms · Standard: 150ms · No large transitions
- Easing: cubic-bezier(0.4, 0, 0.2, 1) — sharp ease-out
- Allowed: opacity on page-load section reveals (once), active-state color transitions on nav
- Never: transforms, slides, bounces, scroll-triggered animations, parallax
- prefers-reduced-motion: all transitions set to 0ms

## Content structure (sections)
1. **Hero** — name, one-line tagline, contact links, resume CTA
2. **Experience** — chronological internships with company, role, dates, 3 bullets
3. **Skills** — four labeled groups (Languages, Libraries, Analytics, Product)
4. **Education** — UCSD, degree, relevant coursework
5. **Contact** — email + LinkedIn

## Never (this project's kill list)
- No Inter or Roboto as fonts — Instrument Sans only
- No purple-to-blue gradients anywhere
- No glassmorphism / blur backdrop
- No card grid with filled backgrounds — hairline borders only
- No bounce/spring easing
- No cyan-on-dark accent color (accent is green #59d299, not cyan)
- No second accent color — one green, that's it
- No uniform spacing — use the 4px scale with intentional rhythm
- No shadow on content elements — only the nav hairline
